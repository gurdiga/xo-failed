Chestiunea

Examinînd cererea CREDITORULUI cu privire la numirea datei ridicării unor sume sau bunuri

Motivele

remiterea de la <%= text('DEBITOR sau DEBITORI', procedură.debitori) %> către CREDITOR a următoarelor bunuri:

    <% for (var nume in bunuri) { %>
  1. <%= nume %> — <%= bunuri[nume].suma %> <%= bunuri[nume].valuta %>
    2. <% } %>

În termenul acordat, <%= text('DEBITORUL sau DEBITORII', procedură.debitori) %> nu <%= text('a sau au', procedură.debitori) %> executat benevol documentul executoriu.

<% if (amînată) { %>

Ridicarea forţată în cadrul acestei proceduri a fost amînată pînă la <%= ultimaAmînare %> din cauza INTRODUCEŢI CAUZA AMÎNĂRII.

<% } %>

În baza celor expuse şi în conformitate cu art. 8, 10, 11, 22, 155 al Codului de executare RM, dispun

Dispoziţia

  1. Se numeşte ridicarea forţată a bunurilor sus menţionate la <% if (amînată) { %> <%= ultimaAmînare %> <% } else { %> <%= procedură['obiectul-urmăririi']['data-şi-ora-ridicării'] %> <% } %> pentru a fi remise CREDITORULUI. Prezenţa <%= text('DEBITORULUI sau DEBITORILOR', procedură.debitori) %> şi CREDITORULUI este obligatorie.

  2. Se înştiinţează <%= text('DEBITORUL sau DEBITORII', procedură.debitori) %> că neprezentarea <%= text('lui sau lor', procedură.debitori) %> nu va împiedica executarea documentului executoriu. Dacă <%= text('DEBITORUL sau DEBITORII', procedură.debitori) %> nu <%= text('va sau vor', procedură.debitori) %> fi prezent, ori nu <%= text('va sau vor', procedură.debitori) %> permite pătrunderea în încăpere, executorul judecătoresc va efectua pătrunderea forţată în aceasta.

  3. A anunţa organele de poliţie să delege un colaborator al organelor de poliţie care să asigure ordinea publică în cadrul procesului de remitere a bunurilor.

  4. Se propune CREDITORULUI să asigure la data ridicării 2 martori asistenţi, hamali, dacă este cazul, transport, specialişti în deschidere forţată a încăperilor, sau să avanseze aceste cheltuieli executorului pentru a le contracta personal.
